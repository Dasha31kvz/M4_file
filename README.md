# Проект Консольное приложение "Спичечный коробок"
![Логотип игры](https://yaart-web-alice-images.s3.yandex.net/a9a633875eb811f19f8b0a36bbf0eb0d:1)
##Правила 
1. Игра расчитана на одного человека
2. Можно брать от одной до трех спичек включительно
3. Необходимо расчитывать количество спичек
4. Игрок не должен допустить ситуации, когда забирает последнюю спичку
